# reverse-proxy-ui

## Installation (Ubuntu / Debian)

```bash
sudo apt-get update && \
sudo apt-get install -y curl git && \
curl -fsSL https://get.docker.com -o get-docker.sh && \
sudo sh ./get-docker.sh && \
git clone https://github.com/momentum100/reverse-proxy-ui && \
cd ./reverse-proxy-ui && \
sudo ./start.sh
```
